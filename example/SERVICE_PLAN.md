## 1. 서비스 개요

### 1.1 서비스명
**WowMall** (와우몰)

### 1.2 서비스 정의
WowMall은 **맞춤형 인쇄 상품 전문 이커머스 플랫폼**입니다. 판매자(파트너)가 자신만의 온라인 상점을 개설하고, 고객이 직접 디자인한 맞춤 인쇄 상품을 주문할 수 있는 B2B2C 마켓플레이스입니다.

### 1.3 서비스 비전
> "누구나 쉽게 자신만의 인쇄 상품을 만들고 판매할 수 있는 플랫폼"

### 1.4 핵심 가치
| 가치 | 설명 |
|------|------|
| **접근성** | 전문 지식 없이도 웹에서 쉽게 디자인 가능 |
| **차별화** | 판매자별 브랜드 아이덴티티 반영 가능 |
| **품질 보장** | 전문 인쇄 서비스(WowPress) 연동으로 고품질 출력 |
| **효율성** | 자동화된 주문-생산-배송 프로세스 |

## 2. 타겟 사용자 및 페르소나

### 2.1 사용자 유형

| 사용자 유형 | 정의 | 주요 니즈 |
|-------------|------|-----------|
| **플랫폼 관리자** | WowMall 운영팀 | 플랫폼 전체 관리, 파트너 승인, 분쟁 조정 |
| **판매자 (Partner)** | 상점 운영 사업자 | 상점 개설, 상품 관리, 매출 증대 |
| **고객 (Customer)** | 상품 구매자 | 맞춤 상품 제작, 편리한 주문, 품질 보장 |

## 6. 화면 구성

주요 화면 와이어프레임은 draw.io 파일로 제공됩니다. 각 파일을 draw.io 또는 diagrams.net에서 열어 확인하세요.

### 6.1 와이어프레임 파일 목록

| 화면 | 파일 경로 | 설명 |
|------|-----------|------|
| **Partner 대시보드** | [`docs/wireframes/partner-dashboard.drawio`](docs/wireframes/partner-dashboard.drawio) | 매출 현황, 주문 통계, 최근 주문 목록 |
| **상품 관리** | [`docs/wireframes/product-management.drawio`](docs/wireframes/product-management.drawio) | 상품 목록, 검색/필터, 일괄 작업, 페이지네이션 |
| **주문 관리** | [`docs/wireframes/order-management.drawio`](docs/wireframes/order-management.drawio) | 주문 목록, 상태 필터, 검색, 일괄 작업 |
| **Shop 홈** | [`docs/wireframes/shop-home.drawio`](docs/wireframes/shop-home.drawio) | 스토어 헤더, 프로모션 배너, 카테고리, 인기 상품 |
| **디자인 에디터** | [`docs/wireframes/design-editor.drawio`](docs/wireframes/design-editor.drawio) | 도구 패널, 캔버스, 에셋 라이브러리, 속성 패널, 레이어 |

### 6.2 화면별 주요 구성요소

#### 6.2.1 Partner 대시보드
- **헤더**: 로고, 사용자 메뉴
- **사이드바**: 네비게이션 메뉴 (대시보드, 상품관리, 주문관리, 디자인편집, CS관리)
- **통계 카드**: 오늘 매출, 신규 주문, 처리 대기, 방문자 수
- **주간 매출 차트**: 일별 매출 추이 그래프
- **최근 주문 목록**: 최신 주문 5건 표시

#### 6.2.2 상품 관리
- **검색/필터**: 키워드 검색, 카테고리 필터
- **일괄 작업**: 전체 선택, 선택 삭제, 카테고리 일괄 변경
- **상품 테이블**: 체크박스, 썸네일, 상품명, 카테고리, 가격, 상태
- **페이지네이션**: 이전/다음, 페이지 번호, 페이지당 개수 선택

#### 6.2.3 주문 관리
- **기간 필터**: 시작일 ~ 종료일 선택
- **상태 탭**: 전체, 결제대기, 결제완료, 제작중, 배송중, 배송완료
- **일괄 작업**: 상태 일괄변경, 엑셀 다운로드, 송장 출력
- **주문 테이블**: 주문번호, 주문일시, 상품정보, 주문자, 결제금액, 상태

#### 6.2.4 Shop 홈
- **스토어 헤더**: 로고, 스토어명, 검색, 장바구니, 마이페이지
- **히어로 배너**: 프로모션 이미지 및 문구
- **카테고리 섹션**: 카테고리 카드 그리드
- **인기 상품 섹션**: 상품 카드 (이미지, 상품명, 가격, 디자인하기 버튼)

#### 6.2.5 디자인 에디터
- **상단 툴바**: 파일/편집/보기 메뉴, Undo/Redo, 줌, 미리보기, 저장, 주문하기
- **좌측 도구 패널**: 선택, 텍스트, 이미지, 도형, 그리기, 템플릿
- **에셋 라이브러리**: 이미지/클립아트/배경/텍스트 탭, 검색, 업로드
- **캔버스 영역**: 상품 프리뷰, 디자인 영역, 핸들/회전
- **뷰 선택**: 앞면/뒷면/좌측/우측
- **레이어 패널**: 레이어 목록, 가시성/잠금, 순서 변경
- **우측 속성 패널**: 위치/크기, 회전, 투명도, 효과, 색상 조정, 상품 옵션, 가격

## 8. 서비스 운영 정책

### 8.1 회원 정책

| 구분 | 정책 |
|------|------|
| **가입 자격** | 만 14세 이상, 사업자는 사업자등록증 필요 |
| **계정 유형** | 개인 고객, 개인 판매자, 사업자 판매자 |
| **인증** | 이메일 인증 필수, 휴대폰 인증 선택 |
| **탈퇴** | 진행 중인 주문 없을 시 즉시 탈퇴 가능 |

### 8.2 상품 정책

| 구분 | 정책 |
|------|------|
| **상품 등록** | WowPress 연동 상품만 등록 가능 |
| **가격 설정** | 원가 + 마진 자율 설정, 최소 마진율 없음 |
| **카테고리** | 최대 3단계 계층 구조 |
| **삭제** | Soft Delete, 주문 이력 보존 |

### 8.3 주문/결제 정책

| 구분 | 정책 |
|------|------|
| **결제 수단** | 신용카드, 간편결제 (Nicepay) |
| **취소/환불** | 제작 시작 전 전액 환불, 이후 협의 |
| **정산** | 배송 완료 후 D+7 정산 |

### 8.4 고객 지원 정책

| 구분 | 정책 |
|------|------|
| **문의 채널** | 티켓 시스템 (1차: 판매자, 2차: 플랫폼) |
| **응답 시간** | 영업일 기준 24시간 이내 1차 응답 |
| **분쟁 조정** | 플랫폼 관리자 중재 |

### 8.5 콘텐츠 정책

| 구분 | 정책 |
|------|------|
| **저작권** | 고객 업로드 이미지의 저작권 책임은 고객에게 있음 |
| **금지 콘텐츠** | 불법, 음란, 혐오, 타인 권리 침해 콘텐츠 |
| **검수** | 판매자 1차 검수, 필요시 플랫폼 2차 검수 |

## 부록 A: 용어 정의

| 용어 | 정의 |
|------|------|
| **Partner (파트너)** | WowMall에서 상점을 운영하는 판매자 |
| **Store (스토어)** | 파트너가 운영하는 개별 온라인 상점 |
| **Template (템플릿)** | 미리 디자인된 편집 가능한 디자인 파일 |
| **WowPress** | 인쇄 및 배송 서비스를 제공하는 외부 파트너사 |
| **칼선** | 인쇄물 재단 위치를 표시하는 선 |
| **안전선** | 중요 콘텐츠가 잘리지 않도록 하는 내부 여백선 |
| **CMYK** | 인쇄용 색상 모드 (Cyan, Magenta, Yellow, Key/Black) |
| **Soft Delete** | 데이터를 실제 삭제하지 않고 삭제 표시만 하는 방식 |
